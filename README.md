# Logistics Portfolio – MEAN Stack Web Application  

## Overview  
This project is a **Single Page Application (SPA)** built using the **MEAN stack** (MongoDB, Express.js, AngularJS, Node.js). It enables users to sign up and access 8+ logistics services including ocean freight, air freight, inland transportation, warehousing, exhibition cargo, and more. The application demonstrates a modern logistics content management platform with integrated frontend, backend, and database workflows.  

---

## Features  
- **Single Page Application (SPA)** architecture for smooth navigation.  
- **Responsive UI** designed with **Figma prototypes**, implemented using **HTML, CSS, JavaScript, Bootstrap, and AngularJS components**.  
- **User Authentication & Signup** with form validation and secure backend data handling.  
- **MongoDB database integration** for storing user information.  
- **Express.js backend APIs** to manage signup requests and static file serving.  
- **Dynamic services section** showcasing different logistics services with linked login pages.  
- **Integrated Video & Text Editor modules** (bonus features).  

---

## Tech Stack  
- **Frontend:** AngularJS, HTML5, CSS3, Bootstrap, JavaScript, Figma  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **Other:** Mongoose, Body-Parser

## Installation & Setup  

### Prerequisites  
- Install [Node.js](https://nodejs.org/) (v14+)  
- Install [MongoDB](https://www.mongodb.com/try/download/community) and make sure it’s running locally (default port `27017`).  

### Clone the Repository  
```bash
git clone https://github.com/your-username/logistics-portfolio.git
cd logistics-portfolio
```

### Install Dependencies  
```bash
npm install
```

### Start MongoDB  
```bash
mongod
```

### Run the Server  
```bash
node server.js
```
The app will be available at:  
`http://localhost:3000/`
